Four hours into a refactor. Thirty files touched. The agent knows your constraints, your plan, what's done, what's left. Then — context full.

Summarize and lose fidelity. Or keep everything and pay for 200K tokens on every tool call. Both options are bad. Both treat your code like random noise.

ZIP has the same problem. Built in 1989. Compresses each file alone. Doesn't know your 15 TypeScript files share the same imports. Doesn't know your test files mirror your source.

PACT understands what it's compressing. That changes everything.
