You're four hours into a refactor. Claude knows everything — your constraints, your plan, which files are done, which are broken. Then the context window fills up and Claude gets amnesia. Cool.

Your options? Summarize and watch the model forget which files it already fixed. Or keep everything and pay for 200K tokens every time it reads a file. Both options suck. You know it. We know it. Anthropic probably knows it too.

Meanwhile ZIP — the compression format older than most developers using Claude Code — is still compressing each file by itself like it's 1989. It doesn't know your 15 TypeScript files all start with the same imports. It doesn't know anything. It literally cannot read.

So we taught compression to read.

ZIP compresses each file alone like a sociopath. PACT concatenates them into one stream — brotli sees the shared imports across all 79 files and goes "oh, I only need to store that once." Wild concept, apparently.

4.5x

Compression ratio

1 MB down to 221 KB. We sort files by extension before concatenating so similar files sit next to each other. Brotli's window sees the pattern. ZIP could never.

0

Dependencies

Zero. Brotli ships with Node.js. We didn't npm install a single compression library. The entire engine is 250 lines of TypeScript. That's it. That's the tweet.
